# AI Agent Learning

Jupyter notebooks covering AI fundamentals, the Anthropic SDK, and Transformer architecture. A personal study log for building AI agents from the ground up.

## Notebooks

| Notebook | Description |
|----------|-------------|
| `AI基础.ipynb` | Core AI concepts and foundational principles |
| `Anthropic SDK.ipynb` | Hands-on practice with the Anthropic API and Claude models |
| `Transformer.ipynb` | Deep dive into the Transformer architecture |

## Setup

```bash
pip install anthropic jupyter
jupyter notebook
```

Set your API key before running the Anthropic SDK notebook:

```python
import os
os.environ["ANTHROPIC_API_KEY"] = "your-api-key-here"
```
